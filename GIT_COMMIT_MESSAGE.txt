## Features
- User profile photo upload and display
- Organization logo upload and display
- Enhanced part usage workflow with better UX
- Transaction history shows machine names instead of codes

## Backend Changes
- Added profile_photo_url to users table
- Added logo_url to organizations table
- Created image upload endpoints (/uploads/users/profile-photo, /uploads/organizations/{id}/logo)
- Fixed /users/me/ endpoint to return profile_photo_url and full organization
- Updated transaction responses to include machine_name
- Added static file serving for uploaded images

## Frontend Changes
- Created ProfilePhotoUpload and OrganizationLogoUpload components
- Improved PartUsageRecorder: Warehouse → Part → Machine workflow
- Added auto-select for single options
- Filter parts by warehouse inventory (stock > 0)
- Added "Use Part" button to machine cards
- Display machine names in transaction history "TO" column
- Increased header photo/logo sizes (32px → 48px)
- Fixed data format issues in part usage component

## Database Schema
- users.profile_photo_url VARCHAR(500)
- organizations.logo_url VARCHAR(500)
- customer_orders.shipped_by_user_id UUID (FK to users)
