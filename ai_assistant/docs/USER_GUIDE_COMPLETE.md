# AutoBoss AI Assistant - Complete User Guide

## Welcome

The AutoBoss AI Assistant is your 24/7 intelligent troubleshooting companion for AutoBoss net cleaning machines. This guide covers everything you need to know.

## Quick Start

1. **Access**: Click the chat icon (ðŸ’¬) in the bottom-right corner of any ABParts page
2. **Describe**: Tell the assistant about your machine issue in your own words
3. **Select Machine**: Choose which AutoBoss machine you're working with
4. **Follow Steps**: Complete each troubleshooting step and report results
5. **Resolve**: Continue until the issue is resolved or escalate to human support

## Chat Interface

### Opening the Assistant
- **Chat Icon**: Always visible in bottom-right corner
- **Click to Open**: Opens full chat window
- **Stays Available**: Icon remains accessible on all ABParts pages

### Chat Window Controls
- **Minimize**: Reduce to icon while keeping conversation active
- **Maximize**: Restore full window
- **Close**: Hide assistant (conversation is saved)
- **Scroll**: Review previous messages in the conversation

### Sending Messages
- **Type**: Enter your message in the text box
- **Send**: Press Enter or click the send button
- **Multiple Messages**: Send as many messages as needed
- **Edit**: Cannot edit sent messages, but can clarify with new messages

## Machine-Specific Troubleshooting

### Selecting Your Machine

**When You Start Troubleshooting**
- Assistant asks: "Which machine are you working with?"
- You see a list of your organization's registered machines
- Select the specific machine having issues

**Machine Information Used**
- Model (V4.0, V3.1B, V3.0, V2.0)
- Serial number
- Installation date
- Recent maintenance history
- Parts usage patterns
- Operating hours

### Benefits of Machine Context

**Personalized Guidance**
- Model-specific instructions
- Considers your machine's maintenance history
- References recent repairs or part replacements
- Suggests preventive maintenance based on usage

**Example**: If your machine recently had a pump replacement, the assistant knows this and can factor it into troubleshooting.
