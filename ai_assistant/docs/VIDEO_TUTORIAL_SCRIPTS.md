# AutoBoss AI Assistant - Video Tutorial Scripts

## Overview

This document contains scripts for video tutorials demonstrating the AutoBoss AI Assistant features. Each script includes narration, on-screen actions, and visual cues.

*Visual: AI provides step-by-step instructions*

**Narrator**: "This interactive process continues until your issue is resolved. The assistant adapts its guidance based on your responses, ensuring you get the most relevant help."

**Narrator**: "You can minimize the chat window to continue working in ABParts while keeping your conversation active."

## Production Notes

### Video Production Guidelines

**Visual Style:**
- Clean, professional interface recordings
- Clear, high-contrast highlighting of UI elements
- Smooth transitions between sections
- Consistent branding (ABParts colors and logo)

**Audio:**
- Professional voice-over recording
- Clear enunciation
- Moderate pace (not too fast)
- Background music (subtle, non-distracting)

**Accessibility:**
- Closed captions in all supported languages
- Audio descriptions for visual elements
- High-contrast visuals
- Large, readable text

**Technical Specifications:**
- Resolution: 1920x1080 (1080p minimum)
- Frame rate: 30fps
- Format: MP4 (H.264 codec)
- Audio: AAC, 192kbps
- Subtitles: SRT format

### Translation Requirements

Each video should be produced in all six supported languages:
- English (original)
- Greek
- Arabic (RTL layout considerations)
- Spanish
- Turkish
- Norwegian

### Distribution

Videos should be:
- Embedded in ABParts help section
- Available on company YouTube channel
- Downloadable for offline viewing
- Linked from AI Assistant interface
- Included in user onboarding
