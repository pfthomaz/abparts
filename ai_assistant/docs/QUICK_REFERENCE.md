## Voice Input Tips

### Speaking Effectively
1. **Speak clearly** at normal conversational pace
2. **Use short sentences** - break up long descriptions
3. **Pause between thoughts** - helps recognition
4. **Minimize background noise** - find quiet area
5. **Repeat if needed** - assistant will ask for clarification

## Machine Selection Guide

### Finding Your Machine
1. Check serial number on machine nameplate
2. Note machine model (V4.0, V3.1B, V3.0, V2.0)
3. Know machine location/name in your facility

### Machine List Shows
- Model number
- Serial number
- Location/name
- Installation date

### If Machine Not Listed
- Verify machine is registered in ABParts
- Check with your administrator
- Ensure you have access permissions
